Waveform Vertex Shader Pipeline: Implementation Complete

LoopModel now renders waveform envelopes from a fixed cylindrical mesh using a vertex shader fed by a 1D RG16F texture. Audio geometry is no longer regenerated every visual update. Envelope decimation is computed in C++, uploaded through Pixel Buffer Objects, and sampled in the GPU for interpolation and deformation.
